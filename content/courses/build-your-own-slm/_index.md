## What You'll Learn

Your phone's autocomplete is doing math every time you type. This course explains that math — from scratch, in plain language, with code you can run right here.

By the end of this course you'll understand:
- What a language model actually is (not the hype — the math)
- How probability distributions make text generation possible
- How to build a tiny working language model in pure Python

## How It Works

Each lesson is either an **Explainer** (read + think) or a **Lab** (run code, see results). No setup. No installs. Everything runs in your browser.
